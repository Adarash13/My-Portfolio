Hand Gesture Device Brightness Controller

The Hand Gesture Brightness Controller is a project that allows users to adjust screen brightness using hand gestures. It leverages computer vision techniques to detect specific gestures via a webcam and translates them into brightness control commands. This provides a hands-free, intuitive way to control screen settings, enhancing user interaction with devices.
